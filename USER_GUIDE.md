# User Guide

## A rendszer célja

Az alkalmazás egy egyszerű karakter-nyilvántartó rendszer, amely lehetővé teszi karakterek kezelését webes felületen keresztül.

## Elérhető funkciók

### 1. Hero List
A kezdőoldalon megjelenik a rendszerben tárolt karakterek listája.

Itt a felhasználó:
- megtekintheti az összes karaktert,
- kiválaszthat egy karaktert,
- törölhet egy karaktert.

### 2. Hero Creator
A létrehozó oldalon új karakter vehető fel.

Lépések:
1. Nyisd meg a **Hero Creator** oldalt.
2. Add meg a karakter nevét.
3. Mentsd el az új elemet.
4. A karakter ezután megjelenik a listában.

### 3. Karakter törlése
A listanézetben a megfelelő művelettel törölhető egy meglévő karakter.

## Használat menete

1. Indítsd el a backendet.
2. Indítsd el a frontendet.
3. Győződj meg róla, hogy a MongoDB is fut.
4. Nyisd meg a frontend felületét böngészőben.
5. Használd a menüt a lista és a létrehozó nézet között.

## Megjegyzés

A rendszer jelenlegi verziója fejlesztési célú, demonstrációs CRUD alkalmazás.
